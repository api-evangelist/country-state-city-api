# Country State City API (country-state-city-api)

The Country State City API is a global geographic reference dataset and REST API exposing 247+ countries, 5,000+ states / provinces, and 150,000+ cities with ISO 3166 codes, phone codes, capitals, currencies, native names, regions, sub-regions, coordinates, time zones, and flag emoji.
